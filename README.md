# LaundryChoiceApp

衣類の素材表示をもとに、洗濯時の扱い方をざっくり判断するアプリです。洗濯タグの画像から判定する方法と、素材名を直接入力する方法の 2 通りに対応しています。

## アプリの使い方

### できること

- 洗濯表示や品質表示の画像を選んで素材カテゴリを判定する
- 素材名を直接入力して素材カテゴリを判定する
- 判定結果に応じて、洗い方・乾かし方・避けたい行為を確認する

### 使い始める手順

1. 依存関係をインストールします。

```bash
npm install
```

2. `.env` を作成します。

画像から素材判定を使う場合は OpenAI API キーが必要です。手入力だけを使う場合でも、開発環境をそろえるために設定しておくと安全です。

```env
OPENAI_API_KEY=sk-xxxx
OPENAI_MODEL=gpt-4.1-mini
REACT_APP_OPENAI_API_KEY=sk-xxxx
REACT_APP_OPENAI_MODEL=gpt-4.1-mini
```

3. アプリを起動します。

```bash
npm start
```

4. ブラウザで `http://localhost:3000` を開きます。

### 画像で判定する方法

1. 画面上部のファイル選択から、洗濯タグや品質表示の画像を選びます。
2. 選択した画像名とプレビューを確認します。
3. `画像から素材判定` を押します。
4. 判定が終わると、素材カテゴリと洗濯方法が下部の結果欄に表示されます。

画像判定では、次の 4 種類のカテゴリのいずれかに分類されます。

- `delicate`: ウール、シルク、レーヨン、カシミヤなど
- `synthetic`: ポリエステル、ナイロン、アクリル、ポリウレタンなど
- `cotton`: 綿、コットン
- `unknown`: 読み取れない、または上の分類に当てはまらない場合

### 手入力で判定する方法

1. 下段の入力欄に素材名を入力します。
2. `素材判定` を押します。
3. 正規化後の文字列と判定結果が表示されます。

入力例:

- `綿 100%`
- `ポリエステル`
- `wool`
- `シルク混`

### 結果の見方

結果欄では主に次の内容を確認できます。

- `classified`: 判定された素材カテゴリ
- `判定元`: 画像 OCR か手入力テキストか
- `洗濯方法`: 洗うときの基本方針
- `乾燥方法`: 乾かし方の目安
- `NG行為`: 避けたほうがよい扱い

このアプリの判定は、日常利用向けの簡易な目安です。高級衣類、特殊素材、装飾の多い衣類は、実際の洗濯表示やメーカー表示を優先してください。

### うまく判定できないとき

- 画像がぼやけている場合は、明るい場所で撮り直してください
- タグ全体よりも、素材表示の文字がはっきり写るように撮ると精度が上がります
- `unknown` になった場合は、手入力で素材名を試してください
- `REACT_APP_OPENAI_API_KEY が未設定です。` と表示された場合は、`.env` を確認してアプリを再起動してください

## OCR テストスクリプト (`scripts/ocr-test.js`)

OpenAI API を使って画像の洗濯表示を読み取り、素材カテゴリを 1 行で出力します。

### 事前準備

1. 依存関係をインストール

```bash
npm install
```

2. プロジェクトルートに `.env` を作成

```env
OPENAI_API_KEY=sk-xxxx
# 任意（未指定時は gpt-4.1-mini）
OPENAI_MODEL=gpt-4.1-mini

# フロント（App.jsx）でOCR実行する場合はこちらも必要
REACT_APP_OPENAI_API_KEY=sk-xxxx
# 任意（未指定時は gpt-4.1-mini）
REACT_APP_OPENAI_MODEL=gpt-4.1-mini
```

3. 判定したい画像を `scripts/` フォルダに置く

対応形式: `png`, `jpg`, `jpeg`, `webp`, `gif`, `bmp`, `tiff`

### 実行方法

画像を指定して実行:

```bash
npm run ocr:test -- scripts/your-image.jpg
```

画像を指定しない場合:

- `scripts/` 内の対応画像を名前順で 1 つ選んで実行します。

```bash
npm run ocr:test
```

### 出力

最後に素材カテゴリだけを出力します（1行）。

- `delicate`
- `synthetic`
- `cotton`
- `unknown`
